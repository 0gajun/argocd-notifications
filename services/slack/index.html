    Slack¶

    1. Create Slack Application using https://api.slack.com/apps?new_app=1
    2. Once application is created navigate to Enter OAuth & Permissions
    3. Click Permissions under Add features and functionality section and add chat:write:bot scope.
    4. Scroll back to the top, click 'Install App to Workspace' button and confirm the installation.
    5. Once installation is completed copy the OAuth token.
    6. Finally use the OAuth token to configure the slack integration in the argocd-notifications-secret secret: 
    apiVersion: v1
kind: Secret
metadata:
  name: argocd-notifications-secret
stringData:
  notifiers.yaml: |
    email:
      host: smtp.gmail.com
      port: 587
      from: <myemail>@gmail.com
      username: <myemail>@gmail.com
      password: <mypassword>
    slack:
      token: <my-token>
      username: <override-username> # optional username
